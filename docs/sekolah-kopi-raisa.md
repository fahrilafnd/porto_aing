# Dokumentasi Visual Project: Sekolah Kopi Raisa

Website referensi:
https://sekolahkopiraisa.com/

Dokumen ini berbasis tampilan visual website (image-based documentation), sehingga fokus pada komponen UI yang terlihat pengguna.

## 1) Hero / Branding Utama

Teks utama yang tampil di bagian atas homepage menekankan positioning produk:
- Membangun Ekosistem Kopi Berkualitas

Fungsi section:
- Menjelaskan value utama website dalam 1 kalimat.
- Menjadi pintu masuk identitas brand Sekolah Kopi Raisa.

## 5) Struktur Halaman Berdasarkan Tampilan

Berdasarkan komposisi konten di homepage, alur visual utama adalah:
- Hero branding
- Tentang Kami
- Produk Kami
- Kegiatan Terbaru
- Footer (alamat, fitur, media sosial, hubungi)

Fungsi struktur:
- Memudahkan user memahami siapa Sekolah Kopi Raisa, apa yang ditawarkan, dan bagaimana cara terhubung.

## 6) Navigasi dan CTA yang Terlihat

Navigasi utama yang tersedia:
- Beranda
- Tentang
- Produk
- Kegiatan

CTA/aksi pengguna:
- Lihat Selengkapnya (menuju halaman detail Tentang)
- Tautan WhatsApp untuk kontak cepat
- Tautan Instagram dan Facebook

## 7) Ringkasan Nilai dari Sudut Visual

- Website menonjolkan kombinasi edukasi, produk, dan kegiatan komunitas kopi.
- Konten visual mendukung storytelling dari sisi kelembagaan dan aktivitas lapangan.
- Elemen kontak dan sosial media ditempatkan jelas untuk mendorong interaksi.

## 8) Catatan Dokumentasi

- Dokumen ini disusun berdasarkan observasi tampilan publik website pada 16 April 2026.
- URL gambar di atas mengambil sumber langsung dari website dan dapat berubah jika pemilik situs melakukan update aset.
