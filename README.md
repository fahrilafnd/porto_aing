# Portfolio Fahril Afandi

Website portfolio berbasis Next.js untuk menampilkan profil, pengalaman, proyek, dan kontak profesional.

## Dokumentasi Project

- Dokumentasi Sekolah Kopi Raisa: docs/sekolah-kopi-raisa.md

## Menjalankan Project

1. Install dependency:
	npm install
2. Jalankan development server:
	npm run dev
3. Buka di browser:
	http://localhost:3000

## Struktur Utama

- app/page.tsx: Halaman utama portfolio.
- public/: Aset statis seperti gambar.
- docs/: Dokumentasi project tambahan.

## Teknologi

- Next.js
- React
- TypeScript
- Tailwind CSS
